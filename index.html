¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos. Bibliografía MDN

Los metodos de peticion HTTP indican las distintas acciones en las que una persona puede manejar un recurso. Existen varios etodos de peticion como GET que es el metodo donde se solicita una representacion de un recurso especifico, estas solo deben recuperar datos. El otro ejemplo es POST que se utiliza para enviar una entidad a un recurso en específico

Los códigos de estado de respuesta HTTP indican si se ha completado satisfactoriamente una solicitud HTTP específica

Nombre clasificación
